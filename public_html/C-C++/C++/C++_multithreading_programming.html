library

POSIX Threads Library based; requires parallel coding Specified by the IEEE POSIX 1003.1c standard (1995). C Language only Commonly referred to as Pthreads. Most hardware vendors now offer Pthreads in addition to their proprietary threads implementations.

Very explicit parallelism; requires significant programmer attention to detail.

OpenMP Compiler directive based; can use serial code Jointly defined and endorsed by a group of major computer hardware and software vendors. The OpenMP Fortran API was released October 28, 1997. The C/C++ API was released in late 1998. Portable / multi-platform, including Unix and Windows platforms Available in C/C++ and Fortran implementations Can be very easy and simple to use - provides for "incremental parallelism"

OpenMP tutorial: computing.llnl.gov/tutorials/openMP https://computing.llnl.gov/tutorials/openMP/
